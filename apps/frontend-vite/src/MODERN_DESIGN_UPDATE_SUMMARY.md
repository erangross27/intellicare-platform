## 📱 Responsive Design (Mobile-First)

### **Mobile Optimizations**
```css
@media (max-width: 768px) {
  /* Reduced padding for mobile */
  .chat-container { padding: 16px !important; }
  
  /* Stacked form layout */
  .chat-form { flex-direction: column !important; }
  
  /* Larger touch targets */
  .chat-input { font-size: 16px !important; }
  .chat-button { width: 100% !important; }
  
  /* Better bubble sizing */
  .message-bubble { max-width: 85% !important; }
}
```

### **Touch-Friendly Elements**
- Input fields: 16px font (prevents zoom on iOS)
- Buttons: Full-width on mobile
- Touch targets: Minimum 44px height
- Proper spacing for thumbs

## ✨ Animation System

### **Message Animations**
```css
@keyframes fadeInUp {
  from { opacity: 0; transform: translateY(20px); }
  to { opacity: 1; transform: translateY(0); }
}
/* Sequential delays for smooth appearance */
```

### **Loading Animations**
```css
@keyframes spin {
  0% { transform: rotate(0deg); }
  100% { transform: rotate(360deg); }
}

@keyframes pulse {
  0%, 100% { opacity: 1; }
  50% { opacity: 0.8; }
}
```

### **Interaction Animations**
- Button hover: Lift effect (-1px transform)
- Input focus: Blue glow effect
- Form submission: Processing state
- Smooth transitions (0.2s ease)

## 🏥 Healthcare-Specific Design Choices

### **Color Psychology**
- **Light backgrounds**: Reduce eye strain, professional appearance
- **Blue accents**: Trust, reliability, medical professionalism
- **Green actions**: Positive, safe, "go" actions
- **High contrast**: Accessibility compliance

### **User Experience**
- **Calming gradients**: Reduce patient anxiety
- **Clear hierarchy**: Easy scanning and reading
- **Smooth interactions**: Professional feel
- **Consistent spacing**: Organized, clean appearance

### **Accessibility Features**
- High contrast ratios (WCAG 2.1 AA compliant)
- Focus indicators for keyboard navigation
- Screen reader friendly structure
- Touch-friendly mobile design
